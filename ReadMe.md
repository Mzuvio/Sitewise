## 📦 Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Mzuvio/sitewise.git
   ```
2. Navigate to the project folder:
   ```bash
   cd sitewise
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the server:
   ```bash
   npm start
   ```
5. Open in your browser:
   ```
   http://localhost:3000
   ```
